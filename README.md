# airborne
Drone flight management and log analytics

Prerequisites:
	
	MySQL 5.7	
	Python 2.7
	Flask 0.12	
	The following Flask extensions:		
		Flask-Login
		Flask-Bcrypt
		Flask-SQLAlchemy	
		Flask-mysqlclient (might require some manual installation)
		Flask-WTForms
		Flask-Migrate
		Pymavlink
		Flask-GoogleMaps
