Airborne

A drone flight management web application that's free and open-source.

Get Started Now Contribute

Features

Drone log analytics using Dronekit-LA

Airborne is capable of log analysis using dronekit-la. It is a fast, powerful, and open-source tool for analyzing ArduPilot logs.

Flight path tracing using Google Maps

GPS coordinates are extracted from logs you upload, and with the help of Google Maps, you can trace the path your drone took during flight.

Free and open-source
